# APSCHEDULER - РУКОВОДСТВО 
 
ЧТО ТАКОЕ APSCHEDULER? 
 
APScheduler (Advanced Python Scheduler) - это библиотека для планирования задач в Python. Представьте, что вам нужно каждый день в 9 утра отправлять отчёт по почте, или каждые 5 минут проверять новый заказ на сайте, или один раз в месяц делать резервную копию базы данных. APScheduler позволяет запланировать выполнение любого Python-кода на определённое время или с заданным интервалом. 
 
В отличие от системных планировщиков (cron в Linux или Task Scheduler в Windows), APScheduler работает внутри вашего Python-приложения, не требует прав администратора и легко настраивается через код. 
 
### Почему именно APScheduler? 
 
- Работает внутри Python-приложения 
- Не требует прав администратора 
- Поддерживает разные типы планирования: интервалы, задержки, расписание 
- Умеет хранить задачи в памяти, в файле или в базе данных 
- Можно запускать задачи в отдельных потоках 
 
--- 
 
ДЛЯ ЧЕГО ЭТОТ ПРОЕКТ? 
 
Этот проект создан, чтобы на практике изучить планирование задач в Python. 
 
### Что мы делаем: 
 
1. Запускаем задачу с задержкой (через 5 секунд) 
2. Запускаем задачу по интервалу (каждые 3 секунды) 
3. Запускаем задачу по расписанию (каждую минуту) 
4. Останавливаем и запускаем задачи вручную 
 
--- 
 
КАК ЭТО РАБОТАЕТ? 
 
APScheduler использует три основных компонента: 
 
- Триггер (Trigger) - определяет, когда запускать задачу (интервал, время, cron-выражение) 
- Хранилище заданий (Job Store) - где хранить список задач (в памяти, в SQLite, в Redis) 
- Исполнитель (Executor) - как запускать задачи (в том же потоке, в отдельном потоке, в процессе)

ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ 
 
### Пример 1: Задача с задержкой 
 
from apscheduler.schedulers.background import BackgroundScheduler 
import time 
 
def my_task(): 
    print("Задача выполнена!") 
 
scheduler = BackgroundScheduler() 
scheduler.add_job(my_task, 'date', run_date=time.time() + 5) 
scheduler.start() 
 
### Пример 2: Задача по интервалу 
 
scheduler.add_job(my_task, 'interval', seconds=5) 
 
### Пример 3: Задача по расписанию 
 
scheduler.add_job(my_task, 'cron', hour=9, minute=0)  # каждый день в 9:00

ЗАКЛЮЧЕНИЕ 
 
APScheduler - это мощный инструмент для автоматизации задач в Python. Он позволяет заменить системные планировщики и встроить расписание прямо в приложение. Главное преимущество - простота настройки и работа без прав администратора.
